The `@repo/core` package provides essential utility functions.

## Installation

```bash
pnpm add @repo/core
```

## API Reference

### `hello()`

Returns a greeting message.

**Returns:** `string`

**Example:**

```typescript
import { hello } from "@repo/core";

const greeting = hello();
console.log(greeting); // "Hello, World!"
```
